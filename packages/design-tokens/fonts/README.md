# Self-Hosted Fonts

These files are the ATNI Climate Design System's five type families, vendored
from the design handoff. They are served with the application; no external
font service is ever used.

| Family | Files | License |
| --- | --- | --- |
| Spartan MB (rework of League Spartan) | `spartanmb-*.woff2` | SIL Open Font License 1.1 |
| TeX Gyre Heros | `texgyreheros-*.otf` | GUST Font License |
| TeX Gyre Heros Cn | `texgyreheroscn-*.otf` | GUST Font License |
| TeX Gyre Adventor | `texgyreadventor-*.otf` | GUST Font License |
| TeX Gyre Termes | `texgyretermes-*.otf` | GUST Font License |

Both licenses permit redistribution and self-hosting. Follow-ups tracked:

1. Vendor the verbatim license texts (OFL 1.1 and GUST Font License) into this
   directory.
2. Convert the TeX Gyre OTF files to WOFF2 for web delivery (the handoff ships
   them as OTF only); keep the OTF files as the reference copies.
